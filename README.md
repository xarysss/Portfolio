# Portfolio terminal - Titouan Mouysset

Portfolio statique inspire par une experience terminal/code editor.

Repo cible : https://github.com/xarysss/Portfolio

## Fichiers

- `index.html` : structure HTML et intro terminal.
- `styles.css` : themes, syntax highlighting et animations.
- `app.js` : contenu personnel, rendu des sections et interactions.
- `assets/CV_Titouan_Mouysset_Alternance_2026.pdf` : CV alternance.

Ouvre `index.html` directement dans un navigateur, ou lance un petit serveur local pour tester :

```powershell
python -m http.server 4173
```
